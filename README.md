## 📚 About Bodymood 
> Bodymood는 어떤 기능이 있나요? 😆
### 1️⃣ 오늘의 운동 감정 선택
- 완료한 운동 세가지를 찾아 선택하고, 하루동안의 감정을 선택해보세요. <br />

### 2️⃣ 운동 포스터 생성
- 내가 선택한 사진과 함께 운동기록을 포스터 만들어드려요. <br />
